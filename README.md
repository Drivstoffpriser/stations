# stations
A common list of fuel stations for all projects sharing fuel price data 


## Data and updates

Stations can be fetched directly from `stations.json` and updated through by opening a pull request.

## Schema

The schema is defined in `stations.schema.json`. It is a [JSON Schema](https://json-schema.org/) file that defines the structure of the data. It can be used to validate data against the schema, or to generate a JSON schema validator.

Validation is run automatically on pull requests.
